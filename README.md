# Personal Portfolio Website

## Overview

This is my personal portfolio website built to showcase my skills, projects, and growth as an aspiring Full Stack Web Developer. The website serves as a central place where visitors can learn more about me, explore my projects, and get in touch.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Professional hero section with introduction
* About Me section with personal and professional background
* Skills section highlighting technologies learned in Block 1
* Projects section featuring completed web development projects
* Contact section with social media links and contact form
* Accessibility features including skip navigation link
* Automatic dark mode support based on system preferences
* Print-friendly résumé view

## Technologies Used

* HTML
* CSS
* Bootstrap 5
* Bootstrap Icons

## Featured Projects

* Tesla Landing Page Clone
* Netflix Landing Page Clone
* YouTube Homepage Clone
* To-Do App
* Quiz Widget
* iHub Africa Website Prototype

## Project Goals

The goal of this portfolio is to showcase my web development skills, demonstrate my understanding of responsive design principles, and provide a professional online presence for future career opportunities.
